### 3.1. Update `requirements.txt`

If you're adding new modules to the project, follow these steps:

1. Create a local `venv`:

    ```shell
    make venv
    ```

3. Activate the local `venv`:

    ```shell
    source ./venv/bin/activate
    ```

4. Run the `freeze` target:

    ```shell
    make freeze
    ```
